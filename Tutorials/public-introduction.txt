Claude Code 教學資源介紹

本資料夾收錄一系列以繁體中文撰寫的 Claude Code 教學頁面，內容聚焦於 AI 程式設計助手的實際應用，適合用於教育訓練、內部推廣、知識傳遞與新手導入。

內容特色
- 從零開始介紹 Claude Code 的定位、用途與使用情境
- 整理安裝需求與跨平台安裝方式，涵蓋 Windows、macOS 與 Linux
- 提供常用指令、權限模式、快捷鍵與特殊語法的快速說明
- 補充進階功能、整合方式與常見問題，降低導入門檻
- 提供完整版本與精簡版本，方便依不同使用情境選擇

檔案內容

1. claude-code-tutorial.html
完整版本教學頁面，內容詳盡，適合初次接觸 Claude Code 的使用者作為入門教材。

2. claude-code-tutorial-v2.html
第二版教學頁面，保留完整教學架構與版型，可作為後續版本延伸或內容調整基礎。

3. claude-code-tutorial-v3.html
精簡版本教學頁面，將重點濃縮為單頁式速查內容，適合快速閱讀、簡報搭配與列印使用。

適用對象
- 想快速理解 Claude Code 的初學者
- 需要進行 AI 工具導入說明的團隊或講師
- 想建立內部教材、訓練手冊或示範文件的單位

建議用途
- 作為 Claude Code 入門教學講義
- 用於工作坊、培訓課程或內部分享
- 作為團隊導入 AI 輔助開發工具的參考資料
