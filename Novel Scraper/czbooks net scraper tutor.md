# 🍤 甜蝦小說爬蟲 — 操作手冊

> 適用網站：**小說狂人 (czbooks.net)**
> 主程式：`scraper.py`

---

## 📖 快速開始：爬一本新小說

### 步驟一：找到小說的「目錄頁」網址

前往 [czbooks.net](https://czbooks.net/)，搜尋你想爬的小說，進入該小說的**目錄頁**。

目錄頁的網址格式通常像這樣：
```
https://czbooks.net/n/xxxxxxxxx
```

> ⚠️ 注意：是目錄頁，不是某一章的內容頁喔！

---

### 步驟二：修改 `scraper.py` 的設定

用編輯器打開 `scraper.py`，找到最上方的「設定區」（約在**第 26 行**），將 `NOVEL_URL` 改成你要爬的小說目錄頁網址：

```python
# 小說的「目錄頁」網址（不是章節頁喔！）
NOVEL_URL = "https://czbooks.net/n/你的小說代碼"
```

你也可以順便修改輸出檔名（第 29 行）：

```python
# 輸出檔名
OUTPUT_FILE = "你想要的檔名.txt"
```

---

### 步驟三：執行爬蟲

打開 **PowerShell** 或 **終端機**，輸入以下指令：

```powershell
& "C:\Users\F096\AppData\Local\Programs\Python\Python312\python.exe" "c:\Users\F096\ClaudeProjects\Novel Scraper\scraper.py"
```

接著就會開始自動爬取，畫面會顯示每一頁的進度：

```
  [1/654] 正在抓取...
  [1/654] 《小說名稱》第1頁 (1900 字) - OK
  [2/654] 正在抓取...
  ...
```

爬完後，小說會儲存在 `Novel Scraper` 資料夾內的 `.txt` 檔案中。

---

## 🔄 中途斷掉怎麼辦？（續爬功能）

如果爬到一半因為網路問題或其他原因中斷了，**不需要重頭開始**！

打開 `scraper.py`，找到第 **35 行**的 `START_FROM_PAGE`，改成你要繼續的頁碼：

```python
# 從第幾頁開始爬（預設 1 = 從頭開始，若中途斷掉可改成續爬的頁碼）
START_FROM_PAGE = 257  # ← 例如從第 257 頁繼續
```

然後再次執行上面的指令即可，新內容會**自動附加**在原本的檔案後面。

> 💡 爬完記得把 `START_FROM_PAGE` 改回 `1`，方便下次爬新的小說。

## ⚙️ 其他可調整的設定

| 設定項目 | 行數 | 預設值 | 說明 |
|----------|------|--------|------|
| `NOVEL_URL` | 第 26 行 | — | 小說目錄頁的網址 |
| `OUTPUT_FILE` | 第 29 行 | `novel_output.txt` | 輸出的檔案名稱 |
| `DELAY_SECONDS` | 第 32 行 | `3` | 每頁間隔秒數，建議 2~5 秒 |
| `START_FROM_PAGE` | 第 35 行 | `1` | 起始頁碼，用於續爬 |

---

## ❓ 常見問題

### Q：爬取速度可以加快嗎？
A：可以把 `DELAY_SECONDS` 調小（例如改成 `2`），但太快可能會被網站封鎖，建議不要低於 2 秒。

### Q：出現「抓取失敗」怎麼辦？
A：爬蟲有內建重試機制（自動重試 3 次）。如果還是失敗，可能是網路不穩，稍等一下再執行即可。

### Q：可以爬其他小說網站嗎？
A：目前只支援 **czbooks.net（小說狂人）**。其他網站的頁面結構不同，需要另外調整程式。
